About the tool This is a a web-based tool which automates the NGS gene expression process. It automates all the individual scripts used in analysis.The user just needs to give the input data sets and can run the analysis with a single click. Since I am using MySQL in the back-end the user need not handle mySQL queries directly. Moreover,the tool is web-based with all scripts,Bowtie and MySQL running on an Apache server in the back-end the user need not install the above tools.
